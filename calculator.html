Calculator Application

Date

Description

A calculator application that uses Java Swing to make a Graphical User Interface. The application has basic functions such as addition, subtraction, multiplication, and division, as well as enhanced functions such as converting the display into other number systems, i.e. Binary, Hexadecimal, etc.

The program makes use of Object Oriented Programming, using encapsulation and inheritance, Error Handling, and Action Event Handling.
