CSV to HTML Converter

Date

July 21, 2023

Description

User will enter commands through command line window. Various commands allow user to give colors to the rows, strip the quotes in the csv file, enable a header for the html file, set a border for the table and give the name to the html file, or report the line count of the file, all optionally. The executable program will take user commands and convert the CSV files (the top left window in the picture above) into HTML files (the top right window) accordingly.

This program demonstrates my ability to manipulate complex and nested conditional structures to meet with the specific features required. It also shows my skills in working with command line arguments, reading from CSV files, and writing the html elements and contents dynamically.
