Inventory User Interface

Date

July 26, 2023

Description

A User Interface that allows user to enter information of three different kinds of inventory: Video Device, HDD, and SSD. This project demonstrates my ability to use classes and child classes to utilize the parent class members and concepts of inheritance in JavaScript. It also shows the use of events that takes user input into various form controls such as textboxes, checkboxes, and drop-down lists. Through button clicked event, the information will be stored into the backend containers such as object arrays.

Technologies

HTML, CSS, JavaScript, JQuery
